== Simple Pet Records 

A web application that provides pet owners the ability to record all their pet information online.

== MVP
The MVP will provide the pet owners the following:

- Signup
- Login
- Pet Owner profile
- Pet profile
- Vet profile
- Pet records
- Reminders
